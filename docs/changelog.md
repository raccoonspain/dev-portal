## 2026-06-27 — Починен надёжный перезапуск systemd

- `Restart=always` вместо `on-failure` — перезапуск при любом коде выхода
- `KillMode=control-group` — systemd убивает все процессы cgroup, а не только main PID
- `StartLimitIntervalSec=0` — снято ограничение на частоту перезапусков
- `TimeoutStopSec=10` — SIGKILL через 10 сек если процесс не завершился
- `server.js`: обработчик `EADDRINUSE` → `process.exit(1)` вместо тихого выхода с кодом 0
