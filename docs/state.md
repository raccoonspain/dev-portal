## Где мы сейчас

Добавлен дашборд ресурсов сервера — вкладка «Сервер» на странице dev-portal. Показывает CPU, RAM, диск и полную таблицу процессов с сортировкой. Блокеров нет.

## Что реализовано

- Навигация: dev-portal · Проекты · Шаблоны · Журнал · Чат Claude
- **Сайдбар:** Чат Claude сворачивается/разворачивается стрелкой (изначально свёрнут); кнопка «Выйти» зафиксирована внизу (position: fixed)
- **dev-portal:** копируемый путь, браузер файлов, VS Code; вкладки **CLAUDE.md / Состояние / История / Решения / Сервер**
- **Дашборд Сервер:** карточки CPU/RAM/Диск с прогресс-барами (зелёный/жёлтый/красный), таблица всех процессов, сортировка по CPU%/MEM%/COMMAND, ручное обновление
- **Проекты:** панель подробностей ❓ — открывает CLAUDE.md и журнал любого проекта прямо в портале
- Файловый браузер: типы `project`, `template`, `portal`; дерево + просмотр файлов
- Журнал проекта: выбор проекта, markdown-рендер 4 вкладок, добавление записей в changelog
- Чат с Claude: SSE-стрим, история через `--resume`, сворачиваемые темы в сайдбаре
- **Systemd:** `Restart=always`, `KillMode=control-group`, `StartLimitIntervalSec=0`; server.js выходит с кодом 1 при EADDRINUSE
