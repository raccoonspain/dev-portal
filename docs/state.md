## Где мы сейчас

Портал полностью работает. Реализованы: таблицы проектов и шаблонов с обзором файлов, открытие в VS Code, скачивание ZIP, страница журнала проекта, страница «🅰 dev-portal» (путь + файловый браузер + журнал самого портала). Сервер запущен через node напрямую (PID в `portal.pid`) — systemd не перезапускается без sudo.

## Что реализовано

- Навигация: dev-portal · Проекты · Шаблоны · Журнал · Чат Claude
- Вкладка **dev-portal**: копируемый путь `/home/deploy/dev-portal`, браузер файлов, кнопка VS Code, встроенный журнал (Состояние / История / Решения)
- Файловый браузер: поддерживает типы `project`, `template`, `portal`; дерево файлов с просмотром содержимого
- Журнал проекта: выбор проекта, markdown-рендер 4 вкладок, добавление записей в changelog
- Чат с Claude: SSE-стрим, история через `--resume`, темы/группы в сайдбаре
- API `/api/portal/{open,tree,file,journal}` — выделенное пространство для портала (D-005)
